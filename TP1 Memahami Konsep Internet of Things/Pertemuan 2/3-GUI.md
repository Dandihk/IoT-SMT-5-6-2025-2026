# Oled 0.9 inch
Layar OLED 0.96 inch adalah salah satu modul layar paling populer untuk ESP32 karena bentuknya kecil, hemat daya, dan sangat tajam. Layar ini biasanya memiliki resolusi 128x64 piksel dan menggunakan protokol komunikasi I2C.

![alt text](image-2.png)
## Spesifikasi:
- Ukuran Layar: 0.96 inch.
- Resolusi: 128 x 64 piksel.
- Driver Controller: SSD1306.
- Tegangan Kerja: 3.3V hingga 5V (Aman untuk ESP32).
- Komunikasi: I2C (Hanya butuh 4 kabel).

## Skema Koneksi ke ESP32
Karena menggunakan I2C, kita menggunakan dua pin khusus pada ESP32 yaitu SDA (Data) dan SCL (Clock)

VCC	3.3V <br>
GND	GND	<br>
SCL	GPIO 22	<br>
SDA	GPIO 21 <br>

## Library
Untuk menggunakannya di esp32 perlu menggunakan library **U8G2 by oliver** atau adafruitGFX

void draw(const char * text) {
    u8g2.clearBuffer(); // untuk menghapus tampilan terlebih dahulu
    u8g2.setFontMode(1);
    u8g2.setBitmapMode(1);
    u8g2.setFont(u8g2_font_profont17_tr);
    u8g2.drawStr(42, 57, text);

    u8g2.sendBuffer(); // mengirim tampilan ke layar
}
